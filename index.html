Основная цель цветовой модели RGB заключается в восприятии, представлении и отображении изображений в электронных системах, таких как телевизоры и компьютеры, хотя она также использовалась в обычной фотографии. До эпохи электроники за цветовой моделью RGB уже стояла солидная теория, основанная на человеческом восприятии цветов.

.h1

{

color

rgb

255, 0, 0

;

}

/* Красный */

.h2

{

color

rgb

0, 255, 0

;

}

/* Зелёный */

.h3

{

color

rgb

0, 0, 255

;

}

/* Синий */

Представление HSL моделирует способ смешивания различных красок для создания цвета в реальном мире, с измерением светлоты, напоминающим различное количество черной или белой краски в смеси (например, для создания «светло-красного» красный пигмент можно смешать с белая краска; эта белая краска соответствует высокому значению «светлости» в представлении HSL).

.h1

{

color

hsl

0, 100%, 50%

;

}

/* Красный */

.h2

{

color

hsl

120, 100%, 50%

;

}

/* Зелёный */

.h3

{

color

hsl

240, 100%, 50%

;

}

/* Синий */

Веб-цвета - это цвета, используемые для отображения веб-страниц во всемирной паутине, а также методы описания и определения этих цветов. Цвета могут быть указаны в виде триплета RGB или в шестнадцатеричном формате (шестнадцатеричный триплет) или в некоторых случаях в соответствии с их общепринятыми английскими названиями. Цветовой инструмент или другое графическое программное обеспечение часто используется для создания значений цвета. В некоторых случаях шестнадцатеричные коды цветов указываются с использованием начального знака числа (#).

.h1

{

color

#FF0000

;

}

/* Красный */

.h2

{

color

#00FF00

;

}

/* Зелёный */

.h3

{

color

#0000FF

;

}

/* Синий */
